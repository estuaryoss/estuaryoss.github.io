Deployer features

Eureka connector

Register with one of the top registration services on the market: NetflixOSS Eureka

Http auth token

Api access is secured through http auth token

Deploy on: docker / kubernetes

Deploy your environment in docker or in kubernetes with kubectl

Automatic environment expire

Don't worry about environment consuming your resources without being deleted

Flask microservice engine

Being built on one of the best microservice engines, comes with a great deal: easy to add features and maintain

Embedded clean-up schedulers

Don't worry about files and folders being created and never deleted. There are automatic schedulers for everything: delete old resources, cycle docker volumes & networks, environment expire

Lightweight image

Being built on Alpine Docker and using Python3.7 makes the image lighter

JSON logging & Fluentd connector

Unleash the power of Fluentd. Send the logs to Elasticsearch / MongoDB / Hadoop. Visualize them with Kibana / Graylog

Deployer use cases

Testing as a service

Deploy with confidence your test environment, run tests, delete it: everything is clean and new, which will boost your confidence in your app and tests

Debug accelerator

Create an environment, find bug, save your environment in docker registry and then deploy it directly on a developer machine or on a special designated debug machine. No more logs, no more pain: provide the yml file and that's it

Infrastructure used for testing

Build your own test infrastructure using estuary services

Templating

Embed your Jinja2 templates on top of the docker image: use the templates to deploy your debug / test environment
